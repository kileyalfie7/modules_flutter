📁 Cấu trúc Module Theme

1. app_colors.dart - Quản lý màu sắc

Màu chính (Primary, Secondary, Accent)
Màu nền và bề mặt (Background, Surface)
Màu text và trạng thái (Success, Warning, Error)
Màu đặc thù cho DMS (Sales, Inventory, Order, Customer, Report)
Gradient và màu viền

2. text_styles.dart - Định nghĩa kiểu chữ

Tuân thủ Material Design 3 typography
Display, Headline, Title, Body, Label styles
Kiểu chữ custom cho DMS (AppBar, Button, Card, Price, Status)
Font family thống nhất (Roboto)

3. theme_data.dart - Cấu hình ThemeData

Light theme và Dark theme hoàn chỉnh
Cấu hình tất cả component themes:

AppBar, Card, Button (Elevated, Outlined, Text)
Input, Icon, Divider, ListTile
BottomNavigationBar, FloatingActionButton
Switch, Checkbox, Radio, ProgressIndicator

4. theme_extensions.dart - Extension tiện ích

Quick access đến theme properties
Helper methods cho màu sắc
Context-aware color getters

🚀 Cách sử dụng
dart// Trong main.dart
import 'core/theme/theme_data.dart';

MaterialApp(
theme: AppTheme.lightTheme,
darkTheme: AppTheme.darkTheme,
// ...
)

// Trong widget
import 'core/theme/app_colors.dart';
import 'core/theme/text_styles.dart';

Text(
'DMS Title',
style: TextStyles.headlineLarge,
)

Container(
color: AppColors.salesPrimary,
// ...
)

// Sử dụng extension
context.primaryColor
context.salesColor
context.textTheme.titleLarge
