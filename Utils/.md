📁 Module Utils Structure

1. validators.dart - Validation theo chuẩn VN

✅ Số điện thoại VN: Hỗ trợ đầu số 03x, 05x, 07x, 08x, 09x
✅ CCCD/CMND: 9 hoặc 12 số
✅ Mã số thuế: 10 hoặc 13 số
✅ Tên tiếng Việt: Có dấu, unicode chuẩn
✅ Địa chỉ, tiền tệ VNĐ, số tài khoản ngân hàng
✅ Mã khách hàng/nhà cung cấp theo format DMS
✅ Combo validation để kết hợp nhiều rule
🎯 Tính năng nổi bật 2. formatters.dart - Format chuẩn VN (tiếp tục)

⏰ Thời gian tương đối: vừa xong, 5 phút trước, 2 ngày trước
📱 Số điện thoại: 0901 234 567
🆔 CCCD/CMND: 123 456 789 / 123 456 789 012
🏛️ Mã số thuế: 123-456-789 / 123-456-789-012
💳 Tài khoản ngân hàng ẩn: 123\*\*\*789
📦 Trạng thái đơn hàng: Chờ xử lý, Đang giao, Hoàn thành
💰 Phương thức thanh toán: Tiền mặt, Chuyển khoản, COD

3. extensions.dart - Extensions tiện ích
   String Extensions:
   dart"nguyễn văn a".titleCase // "Nguyễn Văn A"
   "Trần Thị B".removeVietnameseAccents // "tran thi b"
   "0901234567".formatPhone // "0901 234 567"
   "Hello world".truncate(5) // "Hello..."
   Number Extensions:
   dart1500000.toCurrency // "1.500.000₫"
   1500000.toCurrencyCompact // "1.5M₫"
   1234.5.toDecimal(2) // "1.234,50"
   DateTime Extensions:
   dartDateTime.now().timeAgo // "5 phút trước"
   DateTime.now().weekdayVN // "Thứ hai"
   DateTime.now().isToday // true/false
   Context Extensions:
   dartcontext.showSuccessSnackBar("Thành công!");
   context.screenWidth // Chiều rộng màn hình
   context.isTablet // true/false
   context.hideKeyboard() // Ẩn bàn phím
